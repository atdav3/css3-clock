# CSS3 Working Clock

Đồng hồ analog và digital hoạt động hoàn toàn bằng CSS3, sử dụng CSS animations và shapes, không cần JavaScript hay hình ảnh.

## Tính năng

- ✅ Đồng hồ analog với kim giờ, kim phút, kim giây
- ✅ Đồng hồ digital hiển thị giờ:phút:giây
- ✅ Hoạt động mượt mà với CSS animations
- ✅ Không cần JavaScript
- ✅ Không cần hình ảnh
- ✅ Responsive design

## Cấu trúc dự án

```
css3-working-clock/
├── src/          # Source files
│   ├── index.html
│   ├── style.css
│   └── script.js
├── dist/         # Distribution files
│   ├── index.html
│   ├── style.css
│   └── script.js
├── README.md
└── LICENSE.txt
```

## Cách sử dụng

1. Mở file `src/index.html` hoặc `dist/index.html` trong trình duyệt
2. Đồng hồ sẽ tự động hiển thị và hoạt động

## Công nghệ sử dụng

- HTML5
- CSS3 (Animations, Transforms, Gradients)
